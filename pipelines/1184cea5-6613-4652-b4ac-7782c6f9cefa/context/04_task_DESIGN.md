## 2. 이번 요구사항

**작업 유형:** 신규 기능

**FE 영향 범위:**
- 로그인 페이지 UI (이메일 + 비밀번호 폼)
- 폼 유효성 검사 (이메일 형식, 빈값)
- 에러 메시지 표시
- 로딩 상태 표시
- 대시보드 헤더에 사용자 정보 + 로그아웃 버튼 추가
- 미인증 사용자 리다이렉트
