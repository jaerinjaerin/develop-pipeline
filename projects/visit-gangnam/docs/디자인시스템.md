## 2. 색상 팔레트

### CSS Custom Properties

```css
:root {
  --primary: #C12A59;        /* 크림슨/마젠타 — CTA, 강조, 하이라이트 */
  --primary-dark: #1B1F3B;   /* 다크 네이비 — 다크 섹션 배경 */
  --secondary: #00C9A7;      /* 틸/민트 — 라벨, 뱃지, 보조 강조 */
  --bg: #FFFFFF;             /* 기본 배경 */
  --surface: #F5F5F7;        /* 서피스/카드 배경 */
  --text: #222222;           /* 기본 텍스트 */
  --text-sub: #666666;       /* 보조 텍스트 */
  --radius: 14px;            /* 기본 border-radius */
}
```

### 색상 사용 맥락

| 색상 | 용도 |
|---|---|
| `#C12A59` (--primary) | CTA 버튼 배경, .hl 클래스 텍스트, 활성 탭 하단 바, 호버 배경 |
| `#E84580` | Hero 타이틀 그라데이션 끝 (primary → #E84580) |
| `#1B1F3B` (--primary-dark) | 축제/행사 다크 섹션 배경, 테마카드 오버레이 |
| `#00C9A7` (--secondary) | eyebrow 라벨 색, 통계 숫자, 태그/뱃지, 풋터 호버 |
| `#000000` | body 배경, 풋터 배경, 히어로 오버레이 |
| `#FFFFFF` | 흰색 섹션, 텍스트(다크 배경 위) |
| `#F5F5F7` (--surface) | 인플루언서 섹션 배경, 호버 배경 |
| `#222222` (--text) | 기본 텍스트 색 |
| `#666666` (--text-sub) | 보조 설명 텍스트 |
| `rgba(255,255,255,0.75)` | 네비게이션 링크 기본 색 |
| `rgba(255,255,255,0.7)` | Hero 설명 텍스트 |
| `rgba(255,255,255,0.6)` | 통계 라벨, 패널 sub 텍스트 |
| `rgba(255,255,255,0.4)` | 풋터 링크 |
| `rgba(255,255,255,0.1)` | 패널 번호 텍스트, 글래스모피즘 보더 |
| `rgba(255,255,255,0.08)` | 글래스모피즘 배경, 풋터 구분선 |
| `rgba(0,201,167,0.12)` | 태그 배경 (secondary 12% 투명) |
| `rgba(0,201,167,0.1)` | 인플루언서 태그 배경 |
| `rgba(193,42,89,0.4)` | CTA 버튼 호버 그림자 |
| `rgba(193,42,89,0.8)` | 갤러리 플레이 버튼 배경 |

### 그라데이션

```css
/* Hero 타이틀 "GANGNAM" */
background: linear-gradient(135deg, #C12A59, #E84580);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

/* Hero 배경 오버레이 */
background: linear-gradient(180deg, rgba(0,0,0,0.2) 0%, rgba(0,0,0,0.6) 70%, rgba(0,0,0,0.9) 100%);

/* 핫플레이스 패널 오버레이 */
background: linear-gradient(transparent 40%, rgba(0,0,0,0.8));

/* 테마카드 오버레이 */
background: linear-gradient(transparent 30%, rgba(27,31,59,0.85));
```
